    成员自我介绍

    会长

    王颖捷

    王颖捷，纽卡斯尔大学硕士/博士，毕业后在伯明翰大学和牛津大学做博士后研究。我目前的研究兴趣是了解遗传性心肌疾病的炎症反应，包括肥厚性和扩张型心肌病（即HCM和DCM）。自2013年来，和志同道合的朋友们一起创办了牛津华人生命科学协会，致力于为在牛津生命科学领域学习工作的学生学者创业者们提供一个交流合作的平台。

    副会长

    杜越

    杜越，牛津大学临床医学系WTP博士后，美国&欧洲基因与细胞基因治疗协会成员， ASGCT卓越研究奖得主。研究领域从兽医到体细胞克隆，从基因编辑到肺部疾病的基因疗法开发与临床转化；曾带领团队开发针对免疫力低下人群的首个抗新冠病毒感染的免疫预防。早C晚A，喜看闲书唱闲歌，且及时当勉励。现任OCLSS副会长，在协会中主要服务支持创业与外联板块发展，力求做好传递者与桥梁，把更多的知识与信息通过OCLSS的窗口与大家分享，很高兴你能来。

    副会长/学生主席

    李奇点

    大家好，我叫李奇点，目前是牛津大学化学系博四，主要研究镧系元素在生物成像方面的应用。旅英7年，在伯明翰大学完成本科，在帝国理工完成硕士学业。去年在华大基因和广发期货做过实习。学习和工作之余，参加过一些创投大赛。平时喜欢看书、学日语，最近还迷上了用RAP学英文。
